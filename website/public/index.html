Project "MrFox"

an IoT project to detect the presence of a fox (or a dog) in the garden using a Raspberry Pi 3B+, a PIR sensor and a Pi noir camera.
Once the PIR sensor will detect movement, a photo will be taken and will be published on the project's page.
A read of the Image will be carried by TensorFlow (Coco SSD model) and an email notification will be sent via Mailtrap SMTP server
a SenseHat will also send Temperature, Humidity and Pressure to ThingSpeak and data are visible on the project's page

  • Project's page

RaspberryPi3+

Connected to the home WiFi router using 802.11n protocol.

SenseHat

The SenseHat is connected on GPIO 40 pins (on a mini black hat hack3r) and will be activated by a Python script who will send the data to ThingSpeak
. The same data are published on a website and can be use via API to get the current values of temperature, humidity and pressure

Pi Noir Camera

The Pi Noir Camera, designed to work also in low light conditions, gets triggered only when the PIR sensor detects movement
The images are stored to Firebase Database

Technologies

Main technologies and tools used for the project

Firebase

The Firebase Realtime Database is a cloud-hosted database.Data is stored as JSON and synchronized in realtime to every connected client.

ThingSpeak

ThingSpeak is the open IoT platform with MATLAB analytics. In this project used to read and write data from SenseHat

TensorFlow

TensorFlow is a free and open-source software library for machine learning.For this project used Coco SSD model

node-red

TNode-RED is a programming tool for wiring together hardware devices, APIs and online services in new and interesting ways.Used in this project to test Tensorflow and alternatives

Mailtrap

Mailtrap is a fake SMTP server for development teams to test, view and share emails sent from the development and staging environments without spamming

Python

In this project the main coding language use for all all the function is Python, while for the website basic HTML, CSS and some JavaScript
